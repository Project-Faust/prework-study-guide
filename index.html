HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • "ul" elements are used to create unsorted lists.
  • "li" elements are used to add lines in the form of bullet points to unsorted lists.

CSS

  • A margin indicates how much space we want around an element.
  • A padding indicates how much space we want around the content inside an element.
  • When creating and adjusting styles, sizes can be measured in pixels or as a percentage.
  • When determining color for elements and styles, hexadecimal values can be used to determine the color.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: adds all files to be staged.
  • git commit: commits added files to be pushed.
  • git push: pushes committed files to the repository.
  • git pull: pulls files to local from a repository.

JavaScript

  • Variables can store many types of data, in the form of integers, strings, booleans, and more.
  • Functions are blocks of code that are created to perform a specific task.
  • Logic is used when writing a function to instruct it on how to carry out its task.
  • Arrays can be used to store multiple data types.
  • Loops can be used to cause a function to repeat under certain paremeters when defined.
  • A variable is a named container that allows us to store data in our code.
    • Control flow is the order in which a computer executes code in a script.
